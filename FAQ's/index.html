Frequently Asked Questions

How do I set up the project locally on my machine?

To set up the project locally, follow these steps:

  1. Clone the repository from GitHub: 
    git clone https://github.com/your-repo-url.git
  2. Navigate into the project directory: 
    cd project-directory
  3. Install the necessary dependencies: 
    npm install
  4. Start the development server: 
    npm run dev
  5. Visit http://localhost:3000 in your browser.

I'm having trouble viewing the navigation menu on mobile. How can I fix this?

If the navigation menu is not appearing properly on mobile, try the following steps:

  1. Ensure the hamburger menu icon is visible and functional.
  2. Check for CSS issues with mobile media queries.
  3. Inspect JavaScript handling for errors using the browser console.
  4. Reinstall dependencies using npm install and restart the development server.
